# Members Only Club Page

this is a odin project exercises where I use authentication and authorization


![alt text](image.png)


## techstack

- postgres (neon)
- express
- pasportjs
- bycrypt

## environment

local
```
DB_HOST=
DB_USER=
DB_DATABASE=
DB_PASSWORD=
DB_PORT=

SECRET=secrets
```
production
```
NODE_ENV=production
DATABASE_URL="neon-connection"
```




## get started

```
npm install
npm run populate
node --watch app.js
```
